*To be completed post-capture*

|-------|----------------|---------|---------|
| `0a07365cc01fa9fc82608ba4019af499` | Go SSH client (automated scanner) | TBD | Short sessions, minimal credential list |
| `eeca2460550b9ded084ecf2f70a75356` | OpenSSH standard client | TBD | Human-like behavior |
| TBD | Mirai/botnet variant | TBD | Default IoT credentials |
| TBD | Hydra | TBD | Large credential lists, regular timing |

**First-hour observation:** HASSH `0a07365cc01fa9fc82608ba4019af499` (Go SSH client) appeared in the first connection within 90 seconds and was responsible for the majority of first-hour traffic. This is a known internet-wide scanning tool.
